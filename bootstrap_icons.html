Bootstrap Icons

1.Embedded

在 bootstarp 找到 icons，點按進去複製 svg 貼上，可以修改 width、height、fill(顏色)

2.Sprite

注意 href 路徑需正確，可以修改 width、height、fill

3.External image

將 SVG 檔案當作一般圖片來使用，圖片很難修改顏色，可以使用產生器，修改 css filter 屬性。

https://isotropic.co/hex-color-to-css-filter/

4.Icon font

當作一般文字使用，可修改 font-size、color。

5.使用偽元素搭配 base64

將 svg 或 png 轉成 base64 文字
