按鈕 (Buttons)

使用 .btn 建立按鈕基礎樣式，接著才能使用其他情境顏色、大小、外框的class...

按鈕標籤

.btn 被設計成需要與 button 元素一起使用。 但是，您也可以在 a 或 input 元素上使用這些 Class（儘管有些瀏覽器可能會使用稍微不同的渲染）。

Link

Outline 按鈕樣式

.btn-outline-* 可以使用外框樣式

尺寸

加入 .btn-lg 或 .btn-sm 來套用尺寸。

禁用狀態

透過將 disabled 布林屬性添加到任何 button 元素中，使按鈕看起來處於禁用狀態。被禁用的按鈕具有 pointer-events: none，以防止觸發滑入以及啟用狀態。

a 不支援 disabled 屬性，所以你必須添加 .disabled 使它在視覺上看起來被禁用。
