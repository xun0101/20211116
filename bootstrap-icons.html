Bootstrap Icons

1.Embeded

在bootstrap找到icons，點按進去複製svg貼上，可以修改height、width、fill(顏色)

2.Sprite

注意href路徑需正確，可以修改height、width、fill

3.External image

將 svg 檔案當作一般圖片來使用，圖片很難修改顏色，可以使用產生器，修改css fillter屬性

https://isotropic.co/hex-color-to-css-filter/

Icon font

當作一般文字使用，可修改font-size color

5.使用偽元素搭配base64

將svg或png轉成base64文字
